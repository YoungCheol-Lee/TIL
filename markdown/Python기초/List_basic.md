# List 정리

## Array vs List
- Array 특징   
1. 사이즈가 고정(RAM 낭비가 심함)
2. Array의 요소간 위치가 서로 옆 주소에 존재
3. 실행 속도가 빠름(장점)
- List 특징
1. 사이즈가 가변(RAM 낭비 감소)
2. List의 요소간 위치가 무작위
